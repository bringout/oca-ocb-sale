# Sales Expense Margin

When re-invoicing the expense on the SO, set the cost to the total untaxed amount of the expense.

## Installation

```bash
pip install odoo-bringout-oca-ocb-sale_expense_margin
```

## Dependencies

This addon depends on:
- sale_expense
- sale_margin
