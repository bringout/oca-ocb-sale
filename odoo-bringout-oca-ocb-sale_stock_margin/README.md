# Sale Stock Margin

Once the delivery is validated, update the cost on the SO to have an exact margin computation.

## Installation

```bash
pip install odoo-bringout-oca-ocb-sale_stock_margin
```

## Dependencies

This addon depends on:
- sale_stock
- sale_margin
