# pos_sale



This module adds a custom Sales Team for the Point of Sale. This enables you to view and manage your point of sale sales with more ease.


## Installation

```bash
pip install odoo-bringout-oca-ocb-pos_sale
```

## Dependencies

This addon depends on:
- point_of_sale
- sale_management
