# CRM Gamification

Example of goal definitions and challenges that can be used related to the usage of the CRM Sale module.

## Installation

```bash
pip install odoo-bringout-oca-ocb-gamification_sale_crm
```

## Dependencies

This addon depends on:
- gamification
- sale_crm
