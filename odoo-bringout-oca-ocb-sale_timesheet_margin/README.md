# Service Margins in Sales Orders


Allows to compute accurate margin for Service sales.
======================================================


## Installation

```bash
pip install odoo-bringout-oca-ocb-sale_timesheet_margin
```

## Dependencies

This addon depends on:
- sale_margin
- sale_timesheet
