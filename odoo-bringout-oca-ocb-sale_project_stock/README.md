# Sale Project - Sale Stock

Adds a full traceability of inventory operations on the profitability report.

## Installation

```bash
pip install odoo-bringout-oca-ocb-sale_project_stock
```

## Dependencies

This addon depends on:
- sale_project
- sale_stock
