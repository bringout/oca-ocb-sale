# Sales - Project


Allows to create task from your sales order
=============================================
This module allows to generate a project/task from sales orders.


## Installation

```bash
pip install odoo-bringout-oca-ocb-sale_project
```

## Dependencies

This addon depends on:
- sale_management
- project
