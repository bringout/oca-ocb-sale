# On site Payment & Picking


Allows customers to pay for their orders at a shop, instead of paying online.


## Installation

```bash
pip install odoo-bringout-oca-ocb-website_sale_picking
```

## Dependencies

This addon depends on:
- website_sale_delivery
- payment_custom
